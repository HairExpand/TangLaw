第一百二十条之一 居丧违制罪

在父母、夫之丧期内，脱去丧服而穿吉服的，或者忘哀作舞乐的，徒三年；作杂戏的，徒一年；听奏乐的，或者参加吉席的，杖一百。

在期亲尊长之丧期内，脱去丧服而穿吉服的，杖一百；大功以下尊长，依次减二等。忘哀作舞乐的，以不应为重论；大功以下尊长，以不应为轻论。卑幼减一等。

本条规定了居丧违制罪。其中，在父母之丧期内，脱去丧服而穿吉服的，或者忘哀作舞乐的，属于十恶之罪中的不孝。

丧期和亲等

丧期

丧期

依礼制，亲属亡故设有服丧期。服丧期内有一系列禁忌规定，如禁止嫁娶、作音乐、作杂戏等。违反服丧期禁忌规定的，将依出礼而入刑的原则受刑事处罚。

父之丧期一般为斩衰三年。为人后者为生父，出赘子、出嫁女、出采女为父，服丧齐衰（不杖期）一年。 母之丧期一般为齐衰（杖期）一年。父亡后服丧齐衰三年；子为父亡后改嫁之母、被父所出之母、随改嫁之母服丧齐衰（杖期）一年；为人后者为生母，出赘子、出嫁女、出采女为母服丧齐衰（不杖期）一年。 夫之丧期为斩衰三年。 期亲之丧期为一年。 大功亲之丧期为九月。 小功亲之丧期为五月。 缌麻亲之丧期为三月。

根据襢制的规定，斩衰三年的丧期，满二十七个月举行襢祭后可除服。因此，各项禁忌规定也仅限在二十七个月内。

亲等

本罪根据亲等关系而确定不同的法定刑，同一等亲可能因尊压、出降而有不同的丧期。

例如，姊的亲等是期亲，出嫁后服制降为大功九月。在在室姊之一年丧期内或者在出嫁姊之九月丧期内，脱去丧服而穿吉服的，依在期亲尊长之丧期内，脱去丧服而穿吉服的规定，处杖一百。

父母

父母，即父和母。

一、父

根据第五十二条的规定，父包括生父和养父，不包括继父。

子、在室女、归家出赘子为父服斩衰三年；为人后者为生父，出赘子、出嫁女、出采女为父，服齐衰不杖期。

二、母

根据第五十二条的规定，母包括嫡母、继母、慈母、养母，不包括庶母慈己者。

父亡之子、在室女为母服齐衰三年；父在之子、在室女为母，子为父亡后改嫁之母、被父所出之母、随改嫁之母，服齐衰杖期；为人后者为生母，出赘子、出嫁女、出采女为母服齐衰不杖期。

嫡母、继母、慈母、养母，父亡后改嫁的，降为期亲；被父所出的，降为无服。

三、其他规定

根据第五十二条的规定，嫡孙承祖的，祖父母同父母，但是适用缘坐的除外。因此，除适用缘坐外，有关父母的规定，祖父母对嫡孙同样适用。

夫

夫，指婚姻关系中，妻、媵、妾的男方，服制为斩衰三年。

1、三月庙见、未庙见、就婚三种夫，都是刑法上的夫。 2、订婚夫，在婚姻关系中属于法律上的夫，但不属于刑法上的夫。

期亲

亲等的一种，指本服丧期一年的亲属，包括齐衰杖期和齐衰不杖期的亲属，如祖父母、妻、子、在室女、伯叔父母、兄弟、在室姊妹、在室姑等。

此外，根据第五十二条的规定，称“期亲”的，曾祖父母、高祖父母同。因此，曾祖父母（齐衰五月）、高祖父母（齐衰三月）在刑法上也属于期亲。

大功

五服亲属中第三等，如孙、从兄弟、姊妹等。

大功亲的丧期为九月。

小功

五服亲属的第四等，如从祖祖父母、从祖兄弟、外祖父母、舅、叔等。

小功亲的丧期为五月。

缌麻

五服亲属中最低的一等，如外孙、族母、族姑、族祖父母、族曾祖父母、甥、妻之父母、夫之舅、婿等。

缌麻亲之丧期为三月。

脱去丧服而穿吉服

脱去丧服而穿吉服，即释服从吉。依丧制要求，不同等级的亲属应穿不同的丧服，以尽丧礼。构成本罪要求满足两个条件，一为脱去丧服，二为穿吉服。仅有脱去丧服的行为，不以犯罪论处。

在父母之丧期内，脱去丧服而穿吉服的，属于不孝。

根据亲等关系，在丧期内脱去丧服而穿吉服有不同的法定刑，具体规定如下表。

玩乐

在丧期内作乐的，也构成犯罪。法条将玩乐分为忘哀作舞乐、作杂戏、听奏乐和参加吉席，不同行为有不同的法定刑。

忘哀作舞乐

舞乐，即舞蹈和音乐。作舞乐，是指弹奏舞乐，或者让别人演奏舞乐。舞乐，如金石、丝竹、笙歌、鼓舞等。法条要求忘哀而作，若系思念亡父、亡母、亡夫而作哀乐，则不构成犯罪。

在父母、夫之丧期内忘哀作舞乐的，徒三年。其中，在父母之丧期内忘哀作舞乐的，属于不孝。

在期亲以下之丧期内忘哀作舞乐的，以不应为罪论。其中期亲之丧期，定不应为重罪；大功以下亲之丧期，定不应为轻罪。不应为罪规定在第四百五十条。

作杂戏

作杂戏，指玩各种游戏，如樗[chū]蒲[pú]、双陆、弹碁、象棋等。

在父母、夫之丧期内作杂戏的，徒一年。

听奏乐

听奏乐，与作舞乐不同。听奏乐，是指遇到别人奏乐而去听奏，并非由自己要求他人演奏。

在父母、夫之丧期内听奏乐的，杖一百。

参加吉席

参加吉席，指参加庆礼酒宴。若参加的宴席不属于吉席，则不构成犯罪。

在父母、夫之丧期内参加吉席的，杖一百。
