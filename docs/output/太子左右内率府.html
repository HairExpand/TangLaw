太子左右内率府

率

　　　　　
职事官
官职编制品阶职责
率一人正四品上掌东宫诸门禁卫之法
副率二人从四品上
　 　 　
流外官
官职编制职责
千牛十六人执细刀、弓箭
备身二十八人宿卫、侍从
主仗六十人守戎服、器物
率

率各一人，正四品上。

隋文帝置左、右内率·副率。领东宫千牛、备身侍奉之事，职拟千牛将军。其备身有：千牛备身八人，掌执千牛刀；备身左右十六人，掌供奉弓箭；备身二十人，掌宿卫侍从。炀帝降内率为正五品，皇朝因加至四品上。龙朔二年改为左、右奉裕率，神龙初复旧。

左、右内率府率之职，掌东宫千牛、备身侍奉之事，而主其兵仗，总其府事。

以千牛执细刀、弓箭，以备身宿卫、侍从，以主仗守戎服、器物。

凡皇太子坐朝，则领千牛、备身之属升殿。若射于射宫，则率领其属以从，位定，千牛、备身奉细弓及矢，立于东阶上，西面；率奉弓，副率奉矢及决拾，北面张弓，左执弣，右执箫以进副率以巾拂矢而进，进讫，各退立于位。及射，左、右内率启其矢中及不中，既事，受亦如之。

副率

副率各一人，从四品上。

副率为之贰。

长史

职事官
官职编制品阶职责
长史一人正七品上掌判诸门禁卫之贰
　
流外官
官职编制
亭长二人
掌固二人
长史

长史各一人，从七品上。

隋置右、右内率，有长史以下等员，无兵曹，皇朝置之。

长史掌判诸曹官吏及千牛、备身之贰，余如左、右率府。
